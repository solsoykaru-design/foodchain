# Интеграция раздела «Меню → Элементы меню»

## Структура

Раздел **Меню → Элементы меню** (`menu_items`) использует три компонента:

| Компонент | Файл | Назначение |
|-----------|------|------------|
| `MenuItemsList` | `src/admin/MenuItemsList.tsx` | Таблица с фильтрами, пагинацией, импортом |
| `MenuItemCard` | `src/admin/MenuItemCard.tsx` | Модальное окно создания/редактирования |
| `MenuPage` | `src/admin/MenuPage.tsx` | Карточная версия меню (альтернативный вид) |

## API Endpoint

### `GET /api/menu-items`

Параметры запроса:

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `category_id` | number | — | Фильтр по категории |
| `store_id` | number | — | Фильтр по филиалу (branch_id) |
| `tech_card_filter` | string | `all` | `all` / `yes` / `no` |
| `type` | string | `all` | `all` / `goods` / `service` |
| `search` | string | — | Поиск по названию, артикулу, штрихкоду |
| `sort_by` | string | `name` | Поле для сортировки |
| `sort_order` | string | `asc` | `asc` / `desc` |
| `page` | number | `1` | Номер страницы |
| `limit` | number | `20` | Элементов на странице (макс. 200) |

Ответ:

```json
{
  "items": [
    {
      "id": 1,
      "name": "Цезарь с курицей",
      "imageUrl": "/uploads/dishes/cezar.jpg",
      "barcode": "460000000001",
      "article": "CEZ-001",
      "netto": 0.250,
      "unit": "г",
      "categoryId": 2,
      "categoryName": "Салаты",
      "type": "goods",
      "isAvailable": true,
      "price": 450.00,
      "cost": 180.50,
      "markup": 59.9,
      "techCardId": 5
    }
  ],
  "total": 150,
  "page": 1,
  "limit": 20,
  "totalPages": 8
}
```

Реализация: `server/index.js:1695-1766`.

## База данных

Таблица `dishes` содержит все поля для элементов меню. Файл миграции: `server/migrations/014_menu_items_dishes_documentation.sql`.

## Маршрутизация

Компонент `MenuItemsList` рендерится при значении `adminPage === 'menu_items'` (`AdminApp.tsx:137`).

Навигация добавлена в боковое меню: `AdminApp.tsx:285`
```
Меню/Clients → Меню → Элементы меню
```

## Подключение новых фильтров

Если требуется добавить новый фильтр:

1. **Фронтенд**: Добавьте состояние в `MenuItemsList.tsx`, передайте его в `api.getMenuItems()`.
2. **Бэкенд**: Добавьте обработку параметра в `server/index.js:1695` внутри `GET /api/menu-items`.
3. **БД**: Убедитесь, что поле существует в таблице `dishes`, или сделайте JOIN.

## Импорт из XLSX/CSV

Реализован на фронтенде через библиотеку `xlsx`. Поддерживаемые колонки в файле:

| Русская | English | Описание |
|---------|---------|----------|
| Название | name | Обязательное поле |
| Цена | price | Число |
| Штрихкод | barcode | Текст |
| Артикул | article | Текст |
| Вес / Выход | weight / netto | Число |
| Ед.изм | unit | г, кг, мл, л, шт |
| Категория | category | Сопоставляется по имени |
| Тип | type | goods / service |
| Себестоимость | cost | Число |

## Базовые действия

1. **Создать**: кнопка `+` → модалка `MenuItemCard`.
2. **Редактировать**: клик на строку таблицы → модалка `MenuItemCard`.
3. **Удалить**: кнопка корзины (одиночная) или выбор строк + массовое удаление.
4. **Импортировать**: кнопка «Импорт» → выбор `.xlsx`/`.csv`.
5. **Фильтровать**: выпадающие списки в верхней панели.
6. **Сортировать**: клик на заголовок колонки.
7. **Искать**: текстовое поле в верхней панели (с debounce 300ms).
8. **Массовые действия**: чекбокс «Все» → выбор → меню «Действия».
