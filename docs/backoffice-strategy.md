# Стратегия масштабного обгона конкурентов в back-office

## Видение

FoodChain становится не «еще одним POS + админкой», а **единой операционной платформой ресторана**:
- Автоматизирует 80% решений управляющего.
- Заменяет iiko/R-Keeper + 1С + CRM + мессенджеры + Excel одним продуктом.
- Работает из браузера, мобильного приложения и через AI-ассистента.

## 4 столпа, по которым обгоняем

### 1. Умный маркетинг и лояльность
**Что у конкурентов:** базовые бонусы, простые рассылки.

**Что делаем лучше:**
- **CDP (Customer Data Platform)** — единый профиль клиента со всеми заказами, отзывами, визитами, предпочтениями.
- **RFM + когортный анализ + LTV прогноз**.
- **Автоматические сценарии**:
  - «Не был N дней» → персональная скидка на любимое блюдо.
  - День рождения → бонус + поздравление.
  - Брошенная корзина → push через 30 мин.
  - Повышение уровня лояльности → VIP-оффер.
- **A/B тестирование акций** — какая акция принесла реальный прирост LTV, а не просто съела маржу.
- **Реферальная программа** с отслеживанием источника.
- **Геймификация**: бейджи, челленджи, достижения, турниры среди гостей.
- **Персональное меню**: показывать/скрывать блюда и цены под сегмент клиента.

### 2. Управление персоналом как у enterprise-HR
**Что у конкурентов:** смены, простая зарплата.

**Что делаем лучше:**
- **График с AI-планированием**: система сама предлагает оптимальное количество сотрудников по прогнозу продаж и загрузке.
- **Расчёт зарплаты любой сложности**:
  - оклад/час + ночные + праздничные;
  - % от выручки / от личных продаж / от чаевых;
  - штрафы/премии;
  - расчёт НДФЛ и взносов (для РФ).
- **KPI сотрудников**: скорость обслуживания, средний чек, конверсия апсейла, рейтинг.
- **Мотивация**: автоматические бонусы за выполнение KPI.
- **Табель учёта рабочего времени** с экспортом в контролирующие органы.

### 3. Финансы и бухгалтерия внутри системы
**Что у конкурентов:** экспорт в 1С, базовые отчёты.

**Что делаем лучше:**
- **Полноценный бухгалтерский учёт**:
  - план счетов;
  - ОСВ (оборотно-сальдовая ведомость);
  - баланс, отчёт о прибылях и убытках, движение денежных средств.
- **Кассовая книга** с кассовыми ордерами.
- **Учёт кредиторов/дебиторов** (поставщики, арендодатели, займы).
- **Управленческий учёт**: P&L по филиалам, категориям, блюдам.
- **Бюджетирование**: план vs факт по статьям расходов.
- **Интеграция с банками**: автоматическая загрузка выписок и сверка.
- **Экспорт в 1С / Бухсофт / МойСклад** — но чтобы малый бизнес мог обходиться без них.

### 4. Мобильность и AI
**Что у конкурентов:** в основном десктопная админка, ограниченная мобильность.

**Что делаем лучше:**
- **Мобильное приложение управляющего** (PWA / Capacitor):
  - push-уведомления о критических событиях;
  - быстрые отчёты;
  - утверждение заказов поставщикам одной кнопкой;
  - фотоотчёты и чеки.
- **AI-ассистент управляющего**:
  - «Почему вчера упала выручка на 15%?»
  - «Сколько и чего заказать у поставщиков на следующую неделю?"
  - Автогенерация еженедельного отчёта с рекомендациями.
- **Voice/чат-интерфейс** для управления заказами и отчётами.

## Уникальные конкурентные преимущества (их нет ни у кого в комплексе)

1. **Автопилот закупок**: прогноз продаж → рекомендация заказа поставщику → утверждение в мобильном приложении → приёмка → корректировка остатков.
2. **AI-копирайтер акций**: система генерирует текст SMS/push на основе сегмента и блюд.
3. **Динамическое ценообразование**: автоматический Happy Hour, цены по сегментам, цены по остаткам.
4. **Полный цикл сотрудника**: найм → график → расчёт зарплаты → мотивация → увольнение в одной системе.
5. **Бухгалтерия без 1С**: для малого и среднего бизнеса — полный учёт внутри FoodChain.
6. **AI-ассистент**: естественно-языковой доступ ко всем данным бизнеса.

## План внедрения по кварталам

### Q1
- RFM + CDP + триггерные кампании.
- Расчёт зарплаты и мотивация.
- Мобильное приложение управляющего (PWA).
- Улучшенные закупки: прайс-листы, сравнение поставщиков.

### Q2
- Внутренняя бухгалтерия: ОСВ, баланс, П&У, кассовая книга.
- Консолидированная отчётность по сети.
- Автопланирование производства.
- Геймификация и реферальная программа.

### Q3
- Маркетплейс интеграций.
- AI-ассистент управляющего.
- Честный знак / ЕГАИС.
- Динамическое ценообразование.

### Q4
- Нативные мобильные приложения (iOS/Android).
- Voice-интерфейс.
- Предиктивная аналитика и рекомендации.
- Подготовка к выходу на enterprise-рынок.

## Метрики обгона

Через 12 месяцев FoodChain должен закрыть:
- 100% функционала iiko/R-Keeper в back-office;
- 100% функционала Poster;
- 90%+ функционала 1С:Ресторан (кроме узкой бухгалтерии);
- уникальные фичи: AI-ассистент, CDP+RFM, автопилот закупок, мобильное приложение управляющего.

## Следующий шаг

Начинаем с **Фазы 1 / Q1**. Первый модуль — **RFM + триггерные маркетинговые кампании**, так как он быстро приносит выручку и заметно отличает от конкурентов.
