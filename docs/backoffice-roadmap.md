# Дорожная карта back-office: обгон конкурентов

Цель: вывести FoodChain на уровень, а затем и выше iiko / R-Keeper / Poster по back-office для ресторанов.

## Фаза 1. Быстрые победы (1–2 недели)

1. **Триггерные маркетинговые кампании + RFM**
   - Сегментация клиентов: Recency / Frequency / Monetary.
   - Автоматические рассылки: день рождения, 30 дней без заказа, повышение уровня лояльности, брошенная корзина.
   - Конструктор условий (кто, когда, какое сообщение, канал: push/SMS/email).

2. **Полноценный расчёт зарплаты**
   - Тарифы по должностям (оклад/час, ставка за смену, % от выручки, бонусы).
   - Импорт смен из графика.
   - Автоматический расчёт за период с детализацией по строкам.
   - Экспорт ведомости в PDF/Excel.

3. **Прайс-листы и сравнение поставщиков**
   - История закупочных цен по каждому товару.
   - Привязка поставщика к товару, рейтинг поставщика.
   - Автоподсказка «заказать у самого дешёвого».

4. **Улучшенная инвентаризация**
   - Печать бланка пересчёта со штрихкодами.
   - Мобильный ввод фактических остатков (PWA/сканер).
   - Итоговый акт с подписями ответственных.

## Фаза 2. Углубление (3–4 недели)

5. **Планирование производства на основе прогноза**
   - Автоматический расчёт необходимых заготовок от продаж за N дней.
   - Автосоздание производственных заданий.
   - Календарь производства.

6. **Внутренняя бухгалтерия без 1С**
   - Оборотно-сальдовая ведомость (ОСВ) по счетам.
   - Баланс, отчёт о прибылях и убытках, движение денежных средств.
   - Кассовые ордера (приходный/расходный).
   - Экспорт проводок в 1С / Бухсофт.

7. **Консолидированная аналитика по сети**
   - Единый дашборд по всем филиалам.
   - Сравнение филиалов по выручке, среднему чеку, фудкосту, персоналу.
   - Drill-down от сети до конкретного чека.

8. **Расширенная лояльность**
   - Гибкие уровни и персональные офферы.
   - Реферальная программа.
   - Геймификация: челленджи, бейджи, достижения.

## Фаза 3. Масштабирование и экосистема (1–2 месяца)

9. **Мобильное приложение управляющего**
   - PWA или Capacitor-приложение: уведомления, быстрые отчёты, утверждение заказов поставщикам.
   - Push-алерты: низкие остатки, списания, превышение фудкоста.

10. **Маркетплейс расширений**
    - Страница магазина интеграций (1С, МойСклад, Bitrix24, Telegram, Яндекс.Маркет и т.д.).
    - Включение/выключение модулей.
    - SDK и документация для сторонних разработчиков.

11. **Честный знак / ЕГАИС**
    - Интеграция с API «Честный знак» для маркировки.
    - ЕГАИС-декларации для алкоголя.
    - Ветис/Меркурий для продуктов животного происхождения (опционально).

12. **AI-ассистент управляющего**
    - Естественно-языковые ответы на вопросы: «почему упала выручка вчера?», «что заказать у поставщиков?».
    - Автогенерация еженедельного отчёта с рекомендациями.

## Приоритет старта

Рекомендуемый порядок:
1. RFM + триггерные кампании (быстро, сильно влияет на выручку).
2. Расчёт зарплаты (HR-блок критичен для сетей).
3. Внутренняя бухгалтерия (позволяет отказаться от 1С для малого бизнеса).
4. Мобильное приложение управляющего (отличает от десктопных конкурентов).
5. Маркетплейс + AI-ассистент (формирует экосистему).

## Метрики успеха

- В back-office появляются все блоки, отмеченные в PDF как «❌» или «○».
- FoodChain закрывает 90%+ функционала iiko / R-Keeper в back-office.
- Появляются уникальные фичи: AI-ассистент, маркетплейс, мобильное приложение.
